# WILD! Healing 2026

Single-page site for **WILD! Healing 2026** — a Restore Forward gathering.
Online Sept 17–18, on-the-land gathering Sept 19 in Ava, New York.

- **`index.html`** — the live site. Fully self-contained: HTML, CSS, fonts, and
  base64-embedded imagery in one file. This is the page served by GitHub Pages.
- `wild-healing-2026-v10.html` — prior version snapshot.

## Hosting

Served via GitHub Pages from the repository root (`index.html`). A custom domain
will be connected via Namecheap; once set, a `CNAME` file at the repo root will
hold the domain.
